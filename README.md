# WebGPU 项目

从 Python 项目迁移到 WebGPU 的示例项目。

## 开发

1. 安装依赖：
\`\`\`bash
npm install
\`\`\`

2. 启动开发服务器：
\`\`\`bash
npm run dev
\`\`\`

3. 构建项目：
\`\`\`bash
npm run build
\`\`\`

## 部署到 GitHub Pages

1. 构建项目：
\`\`\`bash
npm run build
\`\`\`

2. 部署：
\`\`\`bash
npm run deploy
\`\`\`

## 浏览器要求

需要支持 WebGPU 的浏览器：
- Chrome 113+
- Edge 113+
- 在 Chrome 中启用：chrome://flags/#enable-unsafe-webgpu
